Welcome. This page helps you quickly find the correct guide for the case you are handling. Use the sections below to know where to start and what to do next.

1. Search for the issue or request

Use the search bar on the main page to type keywords such as dispute type, surcharge, account update, correction code, service level, or document request.

2. Open the guide that matches your case

Review the card title and short description, then open the guide that best matches the customer’s concern or required action.

3. Follow the guide step by step

Answer each part carefully and follow the process flow until you reach the correct action, decision, or next handling step.

4. Use the result to complete the case correctly

Apply the correct disposition, update, correction, validation, or communication based on the guide result.

Quick tips
  • Use specific keywords when searching so you can find the right guide faster.
  • If more than one guide looks similar, open the one with the description that best matches the request.
  • Use the feedback button if you notice a missing step, unclear logic, or something that needs improvement.
  • Use the clock in the header if you need to check Manila time while working.
Need more support?

If the case still does not match any guide or you are unsure which path to take, contact your team lead or EBS Operations support for clarification.
