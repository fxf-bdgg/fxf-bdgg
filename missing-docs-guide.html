A–J Guideline ×
A. REBILL

A1. Document WITH FedEx Stamp:

✔ If document shows debtor’s name:
“The document on file was the only document provided to us at the time of pickup and was used to bill the PRO.”
✔ If document has NO debtor name:
“The document on file was the only document provided to us at the time of pickup. Please reach out to the shipper who may be able to provide a copy of an actual BOL.”
✔ If customer is the shipper:
“The document on file was the only document provided to us at the time of pickup. A letter of authorization with a letterhead is needed from a different debtor accepting the shipping charges or an LOA from the new debtor's email domain.”

A2. NO FedEx Stamp OR No Document:

    📩Follow Sending Emails – Steps 1, 2 & 3
✔ If customer wants rebill per BOL but BOL is missing:
  • Utilize Previous Shipments
  • Check using:
    • Shipper Code
    • Consignee Code
    • Bill-To Code
    • Origin/Destination Center ID
  • Prefer paid/closed PROs
✔ Add keyword:
RVSL
VS-BLER-CASE#-UPDATED BT XXXXXXXX PER PREVIOUS SHIPMENTS
(e.g. PRO XXXXXXX) IN ATTEMPT FOR PAYMENT SINCE NO BOL ON FILE
  • Email Training Team for confirmation

A3. LOA Rebill / Customer Accepting Charges:

  • If rebill per BOL request but BOL is missing → Rebill per LOA without correction fee
  • If rebill per LOA request and BOL missing → Rebill per LOA with correction fee
❗ No need to request BOL imaging
Send LOA to: imaging@fedex.com

A4. If Shipment Exceeds 30 Days During Processing:

  • Utilize Previous Shipments in CAPS
  • Email Training Team for confirmation

A5. If Shipment Becomes Over 90 Days:

  • Proceed with Write-Off
  • Complete Write-Off form
  • Send to Manager for approval
B. CLASS DISPUTE (No ADIM / No CLAS)

B. CLASS DISPUTE (No ADIM / No CLAS)

    📩Follow Sending Emails – Steps 1, 2 & 3
✔ If PRO has NDEN (No Density) keyword:
Advise customer that supporting documents are required.
  • Purchase Order
  • Invoice
  • Catalog Page
  • Specification Sheet
  • Packing Slip
  • Website product specifications
C. CLASS DISPUTE – INSPECTED PRO

C. CLASS DISPUTE – INSPECTED PRO (ADIM / CLAS)

  • Class is valid per inspection
✔ Reply:
“PRO XXXXXXXX has been inspected and found to be at class 300. Please see the attached inspection certificate for your reference.”
  • If customer disputes inspection → Move to Freight Research > Inspection Review
D. WEIGHT DISPUTE (Not Reweighed)
  • 12-digit PRO → Use iSPI Shipment History to verify transmitted weight
  • Non-12-digit PRO → 📩Follow Sending Emails – Steps 1, 2 & 3
✔ If PRO has ADIM:
Check for Dimension Certificate and validate weight details.
  • Move to Freight Research > Inspection Review
E. REWEIGH DISPUTE (With REWE Certificate)
  • 12-digit PRO → Use iSPI Shipment History
  • Non-12-digit PRO → 📩Follow Sending Emails – Steps 1, 2 & 3
✔ If PRO has ADIM:
Verify Dimension Certificate and validate reweigh details.
  • Move to Freight Research > Inspection Review
F. SERVICE TYPE / LEVEL
    📩Follow Sending Emails – Steps 1, 2 & 3

Additionally:

✔ Consignee / 3rd Party / 3PL:
  • Advise BOL not on file
  • Ask them to contact shipper for stamped BOL copy
✔ Shipper:
  • Advise BOL requested from center
  • If they have stamped copy, request submission
✔ If nearing 30 days or multiple attempts:
  • Use Previous Shipments in CAPS
  • If majority moved as ECON or PRTY → Update Service Level
VS-BLER-CASE#-UPDATED TO ECON/PRTY PER PREVIOUS SHIPMENTS
(e.g. PRO XXXXXXX) IN ATTEMPT FOR PAYMENT SINCE NO BOL ON FILE
✔ Illegible BOL:
  • Use Previous Shipments
✔ Provided BOL:
  • Review and process accordingly
G. OVRL / XLGH

If Inspected:

  • Check FACERT
  • Check Shipment Inquiry > Shipment History for inspection photo
✔ If dimension supports fee:
Move to WRO Freight Research > Inspection Review

If NOT Inspected:

    📩Follow Sending Emails – Steps 1, 2 & 3

  • If A0 PRO → Review XREF PROs for documents
    • 📩Follow Sending Emails – Steps 1, 2 & 3
✔ If shipment reaches 90 days without BOL confirming debtor:
Proceed with Write-Off
