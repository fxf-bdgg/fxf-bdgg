A. REBILL

A1. WITH FedEx Stamp

✔ With debtor name → Valid BOL used for billing
✔ No debtor name → Ask shipper for valid BOL
✔ Customer is shipper → LOA required

A2. NO Stamp / No Document

📩 Follow Sending Emails (Steps 1–3)
  • Use previous shipments
  • Check shipper / consignee / bill-to
  • Prefer paid PROs
VS-BLER-CASE#-UPDATED BT XXXXXXXX PER PREVIOUS SHIPMENTS
(PRO XXXXXXX – no BOL on file)

A3. LOA Rebill

  • Rebill per LOA (no correction fee if valid)
  • Send LOA to imaging@fedex.com
B. CLASS DISPUTE
📩 Follow Sending Emails (Steps 1–3)
If NDEN → request supporting documents (Invoice, PO, Spec Sheet, etc.)
C. CLASS DISPUTE – INSPECTED
Class is valid per inspection certificate.
If disputed → Move to Freight Research > Inspection Review
D. WEIGHT DISPUTE
  • 12-digit → check iSPI
  • Non-12-digit → use email guide
If ADIM → validate dimension certificate
E. REWEIGH DISPUTE
📩 Follow Sending Emails (Steps 1–3)
Validate reweigh and dimension certificate
F. SERVICE TYPE
📩 Follow Sending Emails (Steps 1–3)
