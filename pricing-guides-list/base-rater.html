Welcome to the Fedex Queues Decision Guide! This tool is designed to help you quickly access the right guides for handling different queues and processes in FedEx.

  • Navigate the Menu: Click any card to open the corresponding guide, such as Billing Disputes, Pricing, OVC claims, Rebill, or Reweigh cases.
  • Use the Clock & Calendar: Click the clock at the top-right to see the current time and calendar for scheduling or reference.
  • Provide Feedback: Use the comment icon to rate the tool or leave suggestions. Your feedback helps us improve the tool.
  • Quick Tips: Each guide includes step-by-step instructions and decision trees to make handling cases faster and more accurate.

If you encounter any issues or need additional support, please reach out to your team lead or EBS Operations support.

Base Rater Guide

How to access and use the CZAR Rater to verify base rates.

Account Setup

Check your email if you received a message from AdminManager@smc3.com.
Open the CZAR Rater website and follow the instructions to create your password.
Open CZAR Rater
Once the password is changed, try logging in using the new password.
After login, the website will bring you to the Czarlite Web Interface.

How to Use the CZAR Rater Website

Open the PRO in FBI and identify the following information needed for the rater.

City Lookup

Origin – Shipper's ZIP Code
Destination – Consignee's ZIP Code

Class

If Pricing involves FAK Rates, enter the class that applies to the customer's class group.
Example:

If the agreement states that Class 100 applies to shipments with
Class 055, 060, 077, and 090,

Enter Class 100 instead of the actual class listed on the BOL.

Weight

Enter the weight per line item.
If the shipment contains multiple line items, click "Add New Row" and itemize the weight and class per line item.

Rate Family

Select LTL as the Rate Family.

Available Tariffs

Select the rater used in the customer's agreement including the applicable year.
Examples:

FXF 1000 Eff 11/01/2010
or
CZARLITE Eff 02/01/2010
If the correct base rater is not available, send an email to the Training Team / SMEs and include all shipment details so they can verify the correct CWT rate.
